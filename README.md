## steps to run this project <br>
- open terminal in directory you want to git repo on it
  
*run this command in terminal or CMD*
```bash
composer install
```
- create database in your divece by name invoices 

*run this command in terminal or CMD*

```bash
php artisan make:migrate
```
<hr>
- and this
*run this command in terminal or CMD*

```bash
php artisan db seed
```
